# OpenFM

OpenFM is a dashboard for SVXLink and FM-radio networks, styled to be compatible with OpenDVM.  
It is still a work in progress.  
More details will follow.

## Features

- Dashboard interface for SVXLink  
- FM radio network support  
- Style compatibility with OpenDVM
